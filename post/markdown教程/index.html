Markdown语法全面总结

2021-03-05

[TOC]

前言：写过博客或者github的朋友都知道，Markdown语法的重要性，作者总结的日常使用中常用的一些语法知识，快来一起看看吧

1 . 快捷键

功能 快捷键
加粗 Ctrl + B
斜体 Ctrl + I
引用 Ctrl + Q
插入链接 Ctrl + L
插入代码 Ctrl + K
插入图片 Ctrl + G
提升标题 Ctrl + H
有序列表 Ctrl + O
无序列表 Ctrl + U
横线 Ctrl + R
撤销 Ctrl + Z
重做 Ctrl + Y

2 . 基本语法

2.1 字体设置

2.2 分级标题

快捷键是 ctrl +1~4 则是从一级标题到四级标题

2.3 图片链接

快捷键是 ctrl+shift+I

同时支持本地图片和网络上的图片

![图片描述](图片路径)

注：这里的图片描述可以不写。

2.4 分割线

你可以在一行中用三个以上的星号(*)、减号(-)、底线(_)来建立一个分隔线，行内不能有其他东西。你也可以在星号或是减号中间插入空格。

2.5 代码块

直接用~~~即可

还可以选择代码语言的种类

2.6 引用

在被引用的文本前加上>符号，以及一个空格就可以了，如果只输入了一个>符号会产生一个空白的引用。

2.7 列表

（1）无序列表

使用 *，+，- 表示无序列表。
注意：符号后面一定要有一个空格，起到缩进的作用。

（2）有序列表

使用数字和一个英文句点表示有序列表。
注意：英文句点后面一定要有一个空格，起到缩进的作用。

2.8 表格

快捷键ctrl+T快速插入想要生成的表格，还可以直接从网上复制已经做好的表格。
